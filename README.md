Stapler
=======

Stapler combines PDFs using Prawn.

Usage
=====
    # Merge specific PDFs. 
    stapler merge input-1.pdf input-2.pdf output.pdf

    # Merge all PDFs.
    stapler merge *.pdf output.pdf
